# gulp-project
> 用gulp快速搭建你的前端开发环境

![gulp](http://images.cnitblog.com/i/36987/201405/111519280268938.jpg) 

---
##### 用法
1. 安装node（如果被墙，可用[镜像](http://npm.taobao.org/) ）
2. `cnpm install`  安装node依赖包
3. `gulp` 启动开发环境，愉快地工作吧~
4. `gulp build` 生产环境，压缩代码
